### 项目说明
> 本项目是纯前端,无后端代码,用了供亚马逊审核sp app注册资格

#### 文件目录介绍
1. public文件夹,存放网站图标
2. assets存放网站需要的各种图片
3. components存放自定义组件,其中存放了图片和文字的组件
4. layout是布局文件,默认设置了一种布局样式,可另增布局,保留了空文件夹
5. router定义了各页面的url
6. utils里提供获取图片的方法
7. views里存放各页面的具体内容
8. .env是多个环境,方便切换

#### 运行打包
1. 本项目使用VUE框架编写,Yarn管理包,VITE打包
2. 打包命令在package.json中可以看到
